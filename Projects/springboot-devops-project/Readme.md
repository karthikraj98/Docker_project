# Spring Boot DevOps CI/CD Project

## Tech Stack
- Spring Boot
- Maven
- Jenkins
- SonarQube
- Docker

## CI/CD Flow
1. Code pushed to GitHub
2. Jenkins pulls code
3. Maven builds artifact
4. SonarQube scans code
5. Docker image built
6. Container deployed automatically

## API Endpoints
- GET /employees
- POST /employees
- GET /employees/health
